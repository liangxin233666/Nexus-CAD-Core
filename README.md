# Nexus CAD Core - 二维样条曲线分析与建模工具

Nexus CAD Core 是一个基于 **C++17** 和 **OpenGL 3.3+** 开发的高性能二维参数化几何编辑工具。它主要用于三次贝塞尔样条曲线（Cubic Bezier Splines）的交互式设计、拓扑编辑以及深度的几何属性分析。

## 1. 项目核心模块说明

项目采用了模块化设计，将数学计算、拓扑结构、几何算法与交互渲染逻辑分离。

###  基础数学库 (`GeoMath.h`)
*   **功能**：提供二维向量 `Vec2` 的基础运算。
*   **技术细节**：
    *   **编译期计算**：大量使用 `constexpr` 关键字，确保如向量加减、点乘等基础运算能在编译期完成优化。
    *   **鲁棒性处理**：针对浮点数比较，手动实现了不依赖标准库的 `constexpr` 绝对值判断，避免了 C++17 标准下 `std::abs` 无法在常量表达式中使用的问题。
    *   **几何原语**：内置叉乘（用于判断点在线段方位）和归一化（用于物理计算）逻辑。

###  样条曲线数据结构 (`BezierSpline.h`)
*   **功能**：管理由多个三次贝塞尔段组成的复合样条线。
*   **关键技术**：
    *   **连续性约束**：支持 **C0 (Free)**、**C1 (Aligned)** 和 **C2 (Mirrored)** 三种拓扑约束。通过维护锚点与控制手柄的镜像关系，确保曲线在连接处的切线与曲率连续。
    *   **De Casteljau 算法**：用于曲线的细分（Subdivision），在不改变曲线几何形状的前提下，通过插值计算增加控制点，提升局部编辑的精细度。
    *   **拉普拉斯平滑**：实现了 `fairCurve` 算法，通过迭代方式将控制点位置向相邻点均值偏移，用于消除曲线的局部应力或尖锐噪声。

###  几何分析引擎 (`CurveProcessor.h`)
*   **功能**：提供复杂的几何特征计算。
*   **核心算法**：
    *   **曲率分析**：通过求取曲线的一阶与二阶导数，计算精确的曲率 $\kappa$ 值，并生成曲率梳（Curvature Comb）。
    *   **等距偏移（Offsetting）**：沿采样点的法线向量进行平移，用于模拟工业制造中的刀具路径或轮廓偏移。
    *   **拐点检测**：使用 **二分查找法（Bisection Method）** 查找一阶导与二阶导叉乘结果变号的位置，精确锁定曲线的曲率反转点。
    *   **凸包计算**：采用 **Andrew 算法（Monotone Chain）**，在 $O(n \log n)$ 复杂度下计算所有控制点的外接最小凸多边形。
    *   **最近点投影**：结合“全局采样”与“局部二分搜索”，计算鼠标指针到样条线上最近的参数 $t$ 位置。

###  交互状态管理 (`CurveEditor.h/cpp`)
*   **功能**：负责处理用户输入、点位捕捉状态、选择集合以及 UI 状态机。

## 2. 软件主要功能

1.  **交互式建模**：
    *   鼠标左键点击空白处：在末尾追加新的样条分段。
    *   鼠标左键拖拽：调整锚点或控制手柄的位置。
    *   鼠标右键点击：取消当前选中状态。
2.  **高级分析可视化**：
    *   **曲率热力图**：根据局部曲率大小实时变换曲线颜色，通过色彩直观反映几何质量。
    *   **密切圆（Osculating Circle）**：在鼠标吸附点实时显示该位置的切圆，展示曲率半径 $R = 1/|\kappa|$。
    *   **速矢图（Hodograph）**：将曲线的导数空间映射到独立坐标系，反映速度向量的变化轨迹。
3.  **参数化控制**：
    *   实时调节偏移曲线距离、曲率梳缩放比例等。
    *   一键切换深色/浅色工业级 UI 主题。

## 3. 技术规格与性能优化

*   **高性能渲染**：
    *   使用 OpenGL **多重采样抗锯齿 (MSAA 8x)** 提升线条渲染质量。
    *   所有几何离散化逻辑均经过预分配内存（`reserve`）处理，减少实时编辑过程中的内存抖动。
*   **现代 C++ 特性**：
    *   使用 `std::vector` 连续存储控制点，对 CPU 缓存友好。
    *   广泛应用 `const` 成员函数和 `[[nodiscard]]` 属性，增强代码安全性与编译器优化能力。
*   **UI/UX 优化**：
    *   集成 **Dear ImGui**，通过调整 `FrameRounding`、`WindowRounding` 等参数实现现代化、美观且响应迅速的控制面板。

## 4. 编译与依赖

### 依赖环境
- **编译器**：支持 C++17 的编译器（如 MSVC 2019+, GCC 9+, Clang 10+）。
- **库文件**：
  - [GLFW](https://www.glfw.org/) (窗口管理)
  - [Dear ImGui](https://github.com/ocornut/imgui) (UI 框架)
  - [OpenGL](https://www.opengl.org/) (图形 API)

### 快速控制指南
| 按键 | 功能 |
| :--- | :--- |
| **鼠标左键** | 创建点 / 拖拽点 |
| **鼠标右键** | 取消选中 |
| **空格键 (Space)** | 切换选中点的连续性模式 (C0/C1/C2) |
| **UI 按钮** | 执行平滑、细分或切换分析工具 |
